\doxysubsection{Detailed Description}
\hypertarget{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1189}{}\doxysubsection{\texorpdfstring{detect-\/libc}{detect-\/libc}}\label{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1189}
Node.\+js module to detect the C standard library (libc) implementation family and version in use on a given Linux system.

Provides a value suitable for use with the {\ttfamily LIBC} option of \href{https://www.npmjs.com/package/prebuild}{\texttt{prebuild}}, \href{https://www.npmjs.com/package/prebuild-ci}{\texttt{prebuild-\/ci}} and \href{https://www.npmjs.com/package/prebuild-install}{\texttt{prebuild-\/install}}, therefore allowing build and provision of pre-\/compiled binaries for musl-\/based Linux e.\+g. Alpine as well as glibc-\/based.

Currently supports libc detection of {\ttfamily glibc} and {\ttfamily musl}.\hypertarget{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1190}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1190}

\begin{DoxyItemize}
\item {\ttfamily GLIBC} is a String containing the value "{}glibc"{} for comparison with {\ttfamily family}.
\item {\ttfamily MUSL} is a String containing the value "{}musl"{} for comparison with {\ttfamily family}.
\item {\ttfamily family} is a String representing the system libc family.
\item {\ttfamily version} is a String representing the system libc version number.
\item {\ttfamily is\+Non\+Glibc\+Linux} is a Boolean representing whether the system is a non-\/glibc Linux, e.\+g. Alpine.
\end{DoxyItemize}\hypertarget{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1193}{}\doxysubsubsection{\texorpdfstring{detect-\/libc command line tool}{detect-\/libc command line tool}}\label{dir_82cfc848b8e8dab35f41d853d1d375cf_autotoc_md1193}
When run on a Linux system with a non-\/glibc libc, the child command will be run with the {\ttfamily LIBC} environment variable set to the relevant value.

On all other platforms will run the child command as-\/is.

The command line feature requires {\ttfamily spawn\+Sync} provided by Node v0.\+12+.
