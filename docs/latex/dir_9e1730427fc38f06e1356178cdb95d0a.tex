\hypertarget{dir_9e1730427fc38f06e1356178cdb95d0a_autotoc_md1647}{}\doxysubsection{\texorpdfstring{fsevents}{fsevents}}\label{dir_9e1730427fc38f06e1356178cdb95d0a_autotoc_md1647}
Native access to Mac\+OS FSEvents in \href{https://nodejs.org/}{\texttt{Node.\+js}}

The FSEvents API in Mac\+OS allows applications to register for notifications of changes to a given directory tree. It is a very fast and lightweight alternative to kqueue.

This is a low-\/level library. For a cross-\/platform file watching module that uses fsevents, check out \href{https://github.com/paulmillr/chokidar}{\texttt{Chokidar}}.\hypertarget{dir_9e1730427fc38f06e1356178cdb95d0a_autotoc_md1648}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{dir_9e1730427fc38f06e1356178cdb95d0a_autotoc_md1648}

{\bfseries{Important note\+:}} The API behaviour is slightly different from typical JS APIs. The {\ttfamily stop} function {\bfseries{must}} be retrieved and stored somewhere, even if you don\textquotesingle{}t plan to stop the watcher.
