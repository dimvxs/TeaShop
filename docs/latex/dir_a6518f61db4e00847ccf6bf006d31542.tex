\doxysubsection{Detailed Description}
\href{http://travis-ci.org/nodeca/argparse}{\texttt{}} \href{https://www.npmjs.org/package/argparse}{\texttt{}}

CLI arguments parser for node.\+js, with \href{https://docs.python.org/3.9/library/argparse.html\#sub-commands}{\texttt{sub-\/commands}} support. Port of python\textquotesingle{}s \href{http://docs.python.org/dev/library/argparse.html}{\texttt{argparse}} (version \href{https://github.com/python/cpython/blob/v3.9.0rc1/Lib/argparse.py}{\texttt{3.\+9.\+0}}).

{\bfseries{Difference with original.}}


\begin{DoxyItemize}
\item JS has no keyword arguments support.
\begin{DoxyItemize}
\item Pass options instead\+: {\ttfamily new Argument\+Parser(\{ description\+: \textquotesingle{}example\textquotesingle{}, add\+\_\+help\+: true \})}.
\end{DoxyItemize}
\item JS has no python\textquotesingle{}s types {\ttfamily int}, {\ttfamily float}, ...
\begin{DoxyItemize}
\item Use string-\/typed names\+: {\ttfamily .add\+\_\+argument(\textquotesingle{}-\/b\textquotesingle{}, \{ type\+: \textquotesingle{}int\textquotesingle{}, help\+: \textquotesingle{}help\textquotesingle{} \})}.
\end{DoxyItemize}
\item {\ttfamily r} format specifier uses {\ttfamily require(\textquotesingle{}util\textquotesingle{}).inspect()}.
\end{DoxyItemize}
