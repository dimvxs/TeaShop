Node\textquotesingle{}s {\ttfamily assert.\+deep\+Equal() algorithm} as a standalone module. Exactly like \href{https://github.com/substack/node-deep-equal}{\texttt{deep-\/equal}} except for the fact that {\ttfamily deep\+Equal(\+Na\+N, Na\+N) === true}.

This module is around \href{https://gist.github.com/2790507}{\texttt{5 times faster}} than wrapping {\ttfamily assert.\+deep\+Equal()} in a {\ttfamily try/catch}.
