\doxysubsection{Detailed Description}
Abstract base entity representing a user in the system.

This is the root of a {\ttfamily \doxylink{}{Inheritance\+Type\#\+JOINED}} inheritance hierarchy. Concrete user types such as {\ttfamily Customer} and {\ttfamily Admin} extend this class. Common fields are stored in the {\ttfamily users} table, while type-\/specific columns are stored in their own dedicated tables.

The {\ttfamily \doxylink{}{login}} field is unique across all user types and is used as the primary credential identifier. The {\ttfamily \doxylink{}{password\+Hash}} stores the hashed password (e.\+g. BCrypt). The {\ttfamily \doxylink{}{last\+Activity}} field is updated on every successful login or meaningful action.
