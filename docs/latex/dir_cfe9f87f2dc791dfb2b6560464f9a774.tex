This package has two modes that are used depending on the environment and parameters of use.\hypertarget{dir_cfe9f87f2dc791dfb2b6560464f9a774_autotoc_md344}{}\doxysubsubsection{\texorpdfstring{old}{old}}\label{dir_cfe9f87f2dc791dfb2b6560464f9a774_autotoc_md344}

\begin{DoxyItemize}
\item Node.\+js below {\ttfamily 10.\+10} or when the {\ttfamily stats} option is enabled
\end{DoxyItemize}

When working in the old mode, the directory is read first ({\ttfamily fs.\+readdir}), then the type of entries is determined ({\ttfamily fs.\+lstat} and/or {\ttfamily fs.\+stat} for symbolic links).\hypertarget{dir_cfe9f87f2dc791dfb2b6560464f9a774_autotoc_md345}{}\doxysubsubsection{\texorpdfstring{modern}{modern}}\label{dir_cfe9f87f2dc791dfb2b6560464f9a774_autotoc_md345}

\begin{DoxyItemize}
\item Node.\+js 10.\+10+ and the {\ttfamily stats} option is disabled
\end{DoxyItemize}

In the modern mode, reading the directory ({\ttfamily fs.\+readdir} with the {\ttfamily with\+File\+Types} option) is combined with obtaining information about its entries. An additional call for symbolic links ({\ttfamily fs.\+stat}) is still present.
