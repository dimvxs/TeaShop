A tiny, secure, URL-\/friendly, unique string ID generator for \+Java\+Script.

\begin{quote}
“\+An amazing level of senseless perfectionism, which is simply impossible not to respect.\+” \end{quote}



\begin{DoxyItemize}
\item {\bfseries{Small.}} 130 bytes (minified and gzipped). No dependencies. \href{https://github.com/ai/size-limit}{\texttt{Size Limit}} controls the size.
\item {\bfseries{Fast.}} It is 2 times faster than UUID.
\item {\bfseries{Safe.}} It uses hardware random generator. Can be used in clusters.
\item {\bfseries{Short IDs.}} It uses a larger alphabet than UUID ({\ttfamily A-\/\+Za-\/z0-\/9\+\_\+-\/}). So ID size was reduced from 36 to 21 symbols.
\item {\bfseries{Portable.}} Nano ID was ported to 20 programming languages.
\end{DoxyItemize}
