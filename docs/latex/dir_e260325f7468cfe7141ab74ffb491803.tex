The {\ttfamily retry()} method will either pass through the result on success or wait and retry on failure. Any error that isn\textquotesingle{}t caught by the retrier is automatically rejected so the end result is a transparent passing through of both success and failure.\hypertarget{dir_e260325f7468cfe7141ab74ffb491803_autotoc_md290}{}\doxysubsubsection{\texorpdfstring{Setting a Timeout}{Setting a Timeout}}\label{dir_e260325f7468cfe7141ab74ffb491803_autotoc_md290}
You can control how long a task will attempt to retry before giving up by passing the {\ttfamily timeout} option to the {\ttfamily Retrier} constructor. By default, the timeout is one minute.
