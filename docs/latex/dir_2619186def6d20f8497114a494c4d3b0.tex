JSON functions that can convert buffers!

\href{http://travis-ci.org/dominictarr/json-buffer}{\texttt{}}

\href{https://ci.testling.com/dominictarr/json-buffer}{\texttt{}}

JSON mangles buffers by converting to an array... which isn\textquotesingle{}t helpful. json-\/buffers converts to base64 instead, and deconverts base64 to a buffer.
